## Fullstack preset

* NestJS
* React
* Typescript
* SCSS
* Webpack
* Babel
* ESLint
* Prettier
* Docker
* nginx
* ✯Bonus✯ Theme switch

### Run in Development mode
Client
```shell
yarn open:dev
yarn dev:client
```
Server
```shell
yarn dev:server
```

### Run in Production mode
```shell
docker-compose up -d --build
yarn open:prod
```
